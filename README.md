# AutoBackup Vault 🗂️
### Smart File Backup and Recovery System

## Description
AutoBackup Vault automates file backups and allows easy recovery.
It creates logs for all actions and keeps files safe.

## Tools & Technologies
- Bash Shell Scripting
- Git & GitHub
- macOS Terminal (Linux compatible)

## How to Run
```bash
chmod +x autobackup.sh
./autobackup.sh          # Run backup
./autobackup.sh --restore # Restore files
